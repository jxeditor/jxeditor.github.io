注意

本地IDEA运行无效,打包到集群上运行.

这只是将数据写入HDFS上,相当于往Hive在Hdfs上的目录底下直接上传格式化好的文件,这个时候查询Hive表,是不会出现数据的.

需要执行 msck repair table tableName; 命令修复分区.

或者执行 alter table tableName add partition(分区字段=’值’); 添加分区

瑕疵

需要人为修复或者脚本修复Hive的分区,完美方案应该将这种修复放入代码实现.
